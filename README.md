# Home SOC Lab

## Overview
This project documents my personal Security Operations Center (SOC) lab environment.

The goal of this lab is to practice:

- Security monitoring
- Log analysis
- Threat detection
- Incident response

## Tools Used

- Wazuh
- Kali Linux
- Windows Server
- VirtualBox
- Sysmon

## Lab Architecture

SOC Lab Environment:

Attacker Machine:
- Kali Linux

Target Machine:
- Windows Server with Sysmon

Monitoring:
- Wazuh SIEM

## Objectives

- Detect brute force attacks
- Monitor suspicious logins
- Analyze security alerts
- Practice incident response

## Skills Practiced

- SIEM monitoring

- Network security
